This is a water-feeding system which can be controlled from anywhere and at any time comfortably and quickly without any confusion and doubt. Remote usage with large range of operability for the system is achieved by GSM network, which is also highly reliable in the current era of ubiquitous wireless communication. The range of GSM is literally the whole globe, therefore the garden can be maintained well even when on international travel.


-----------------Components--------------------

TM4C123GH6PM microcontroller
SIM900 GSM module
Nokia-5110 monochrome 48 Ã— 84 pixels matrix LCD display
Solenoid based electrically controllable valves
Soil Moisture Sensors (any basic model)


----------------Code----------------------------

"smartSprinkler-s2.ino" contains the complete functional code for execution on TM4C123GH6PM microcontroller. It is written using Energia IDE for Stellaris boards and system debugging is done using the serial monitor in Energia. "LCD_5110_SPI" library provided in "Nokia 5110" folder should be added into energia libraries for the functioning of the Nokia LCD display.
